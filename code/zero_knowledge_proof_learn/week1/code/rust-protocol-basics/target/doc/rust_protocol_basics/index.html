Rust Protocol Basics - Core cryptographic and serialization utilities

This library provides fundamental building blocks for cryptographic protocols and secure communication systems. It demonstrates Rust’s ownership system, error handling patterns, and type safety features.

§Quick Start

use rust_protocol_basics::*;

// Serialize a struct and hash it
let data = ExampleStruct::new(1, "test".to_string(), vec![1, 2, 3]);
let bytes = data.to_bytes().unwrap();
let hash = sha256(&bytes);
let hash32 = Hash32::new(hash);

println!("Hash: {}", hash32);
