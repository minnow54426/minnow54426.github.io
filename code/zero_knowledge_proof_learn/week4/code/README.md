# toychain-rs

A minimal blockchain state transition function implementation in Rust.

## Overview

This project implements account-based blockchain state management with transaction validation and block application.

## Dependencies

- `tx-rs` - Transaction types from Week 3
- `ed25519-dalek` - Cryptographic signatures
- `sha2` - SHA-256 hashing
- `serde` - Serialization framework

## Development

```bash
# Build the project
cargo build

# Run tests
cargo test

# Run clippy
cargo clippy
```
